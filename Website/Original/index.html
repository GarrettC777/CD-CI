Our choice in project has much more meaning than our choice in name luckily. A staging Environment is a useful tool to test out new configurations without disrupting the live website. Amazon has created resources which can be used injunction to automate the deployment changes to the staging website as well as the live website.

  • AWS EC2 is a web service that provides secure, resizable compute capacity in the cloud.
  • AWS CodeCommit is a source control service that hosts secure Git-based repositories.
  • AWS CodeDeploy is a deployment service that automates software deployments to a variety of compute services.
  • AWS CodePipeline is a continuous delivery service that helps you automate your release pipelines for fast and reliable updates.

However, our project was focused on automating the creation of these resources using terraform. Terraform is a tool that provides infrasructure as code which along with AWS CLI allows us to automate the CI/CD Pipeline creation.
